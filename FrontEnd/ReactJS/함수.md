# 함수
### modifire
+ 값을 부여하면 그 값으로 업데이트하고 리렌더링을 일으킴
+ ex
    ```js
    const [counter, modifire] = React.useState(0);
    function OnCounter(){
        modifire(counter + 1);
    }
    return(
        <h1>counter</h1>
    );
    ```
    
    ```js
    const [counter, setCounter] = React.useState(0);
    ```
    보통 앞에는 counter처럼 쓰고싶은 이름을 쓰고 뒤에는 setCounter처럼 set뒤에 데이터 이름을 붙여준다고 한다
