### 단어 뜻
        origin : 원격 저장소의 이름이다
        master : 브랜치중에 가장 기본적인 브랜치이다
        remote : 인터넷이나 네트워크 어딘가에 있는 저장소

        check out : 전환하기

        clone : 복제하기

        push : • 원격 저장소로 변경된 파일을 업로드하는 것

        commit : 추가하기

        add : • git add <인덱스에 등록할 파일 명> → 변경 사항을 인덱스에 등록

        pull : 가져와 병합하기

        git : init 생성하기

        branch : 독립적으로 어떤 작업을 진행하기 위한 것
