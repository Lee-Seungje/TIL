# REST

자원을 이름으로 구분하여 해당 자원의 상태를 주고받는 모든 것

-   HTTP URI를 통해 자원을 명시
-   HTTP Method를 통해 해당 자원에 대한 CRUD Operation을 적용

#### CRUD

-   Create
-   Read
-   Update
-   Delete

### 구성 요소

-   자원 Resource : HTTP URI
-   자원에 대한 행위 Verb : HTTP Method
-   자원에 대한 행위의 내용 Representations : HTTP Messege Pay Load

### 특징

-   Server-Client
-   StateLess
-   Cacheable
-   Layered System
-   Uniform Interface
