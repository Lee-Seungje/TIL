# Create React App

아래와 같은 코드를 콘솔에 입력해준다

```cmd
npx create-react-app NameForReactApp(만들 폴더의 이름)
```

package.json을 확인한다\
package.json에는 우리가 쓸 수 있는 script들이 있다\
이것들은 create-react-app 팀에 의해서 만들어진 것이다

```cmd
npm run
```

을 입력해보자\
자동적으로 브라우저가 열리게된다

create react app의 장점중에는 auto-reload가 된다는 것이 있는데\
이것은 익스텐션을 깔면 된다
